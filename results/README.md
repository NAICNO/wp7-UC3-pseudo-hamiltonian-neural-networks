## Results Directory

This directory stores training outputs, model checkpoints, and evaluation results
generated by the Pseudo-Hamiltonian Neural Networks demonstrator.

Contents after running experiments may include:

- Trained model checkpoints (`.pt` files)
- Training loss curves and metrics
- Evaluation plots (trajectory comparisons, energy conservation)
- Exported figures for publications

This directory is initially empty. Run the demonstrator notebook or
`example_scripts/train_model.py` to generate results.
